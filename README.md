# Premium Static Portfolio

A high-performance, single-file static portfolio website converted from Django. It features a premium "Cybersecurity" theme, smooth animations, and is fully responsive.

## Features

- **Single-File Architecture**: Everything (HTML, CSS, JS) is in `index.html`.
- **Premium Design**: Dark theme with glowing accents and smooth gradients.
- **Interactivity**: 
    - Smooth scroll navigation
    - Mobile-responsive menu
    - AOS (Animate On Scroll) animations
    - Simulated contact form

## How to Run

1.  Simply open `index.html` in any modern web browser.
2.  Alternatively, you can serve it using a local server:
    ```bash
    # using python
    python -m http.server 8000
    ```

## Deployment

This site is ready for static hosting. You can deploy it to:
- **GitHub Pages**
- **Vercel** (just drag and drop the folder)
- **Netlify**
